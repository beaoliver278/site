Circuitos Digitais e Introdução à Linguagem C#

Circuitos Digitais:

Nesta aula, exploramos os fundamentos dos circuitos digitais e introduzimos conceitos básicos da linguagem C#.

Como funcionam os circuitos digitais

Os circuitos digitais têm três componentes principais:

  • Entrada: É onde o circuito recebe informações. Pode ser um sinal de um sensor, um botão pressionado ou outros dispositivos de entrada.
  • Processamento: Nesta fase, o circuito executa operações com base nas entradas. Por exemplo, pode combinar sinais de entrada usando portas lógicas (AND, OR, NOT) ou outras operações lógicas.
  • Saída: O resultado do processamento é fornecido na saída. Pode ser um LED aceso, um motor ligado ou outro dispositivo controlado.

Entender como funcionam entrada, processamento e saída é fundamental para trabalhar com circuitos digitais.

Portas Lógicas e Fórmulas:

As portas lógicas são componentes fundamentais em circuitos digitais. Elas recebem entradas digitais e produzem uma saída com base em uma operação lógica específica. As portas lógicas mais comuns são AND, OR, NOT, NAND, NOR, XOR e XNOR. Cada porta lógica tem uma tabela verdade que mostra todas as combinações possíveis de entradas e suas saídas correspondentes.

ICs com Portas Lógicas

Os circuitos integrados (ICs) contêm várias portas lógicas num único chip. Aqui estão alguns exemplos:

IC 7408

Contém quatro portas AND. Você pode usá-lo para construir circuitos com portas AND.

IC 7432

Contém quatro portas OR. Utilize-o para circuitos com portas OR.

IC 7404

Contém seis portas NOT. Use-o para circuitos com portas NOT.

Introdução à Linguagem C#:

Vamos também dar os primeiros passos na linguagem C#, criando um circuito simples com Arduino.

Código Exemplo:

        
            /*
            -----------------------------
            Nome: Beatriz Oliveira      -               
            Itinerário de Robótica      -
            Curso de Arduino            -
            Edusesc Taguatinga Norte    -
            -----------------------------
            */

            // Declaração de Variáveis e Constantes
            const int LED = 13;

            // Configurações
            void setup(){
                pinMode(LED, OUTPUT);
            }

            // Repetições
            void loop(){
                digitalWrite(LED, HIGH);
                delay(1000); // 1 segundo

                digitalWrite(LED, LOW);
                delay(1000); // 1 segundo
            }

Como o Código Funciona:

  • Declaração de Variáveis e Constantes: O código começa declarando uma constante para o pino do LED (13).
  • Configurações: Na função `setup()`, configuramos o pino do LED como saída usando a função `pinMode`.
  • Repetições: Na função `loop()`, alternamos entre acender e apagar o LED usando `digitalWrite`, com intervalos de 1 segundo (`delay(1000)`).
