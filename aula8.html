Momento 1: Circuitos das Portas Lógicas AND, OR e NOT

Na primeira parte da aula, montamos circuitos utilizando portas lógicas AND, OR e NOT. Utilizamos dois botões como entradas e um LED como saída para cada circuito. Testamos cada circuito de acordo com suas respectivas tabelas verdade, gerando um vídeo para cada porta lógica:

Porta Lógica AND

Funcionamento:

    Quando ambos os botões A e B estavam pressionados (representando entradas HIGH ou 1), o LED acendeu.

    Se qualquer um dos botões, ou ambos, não estavam pressionados (representando entradas LOW ou 0), o LED ficou apagado.

Tabela Verdade AND:

  • A = 0, B = 0: LED apagado.
  • A = 0, B = 1: LED apagado.
  • A = 1, B = 0: LED apagado.
  • A = 1, B = 1: LED aceso.

Porta Lógica OR

Funcionamento:

    O LED acendeu quando pelo menos um dos botões (A ou B) estava pressionado.

    O LED ficou apagado somente quando ambos os botões não estavam pressionados.

Tabela Verdade OR (no vídeo):

    A = 0, B = 0: LED apagado.

    A = 0, B = 1: LED aceso.

    A = 1, B = 0: LED aceso.

    A = 1, B = 1: LED aceso.

Porta Lógica NOT

Funcionamento:

  • O LED acendeu quando o botão não estava pressionado (entrada LOW ou 0).
  • O LED ficou apagado quando o botão estava pressionado (entrada HIGH ou 1).

Tabela Verdade NOT (no vídeo):

  • Entrada = 0: LED aceso.
  • Entrada = 1: LED apagado.

Momento 2: Circuito com LDR e LED

Na segunda parte da aula, montamos um circuito elétrico que utiliza um sensor de luz (LDR) como entrada e um LED como saída, processando o valor analógico no Arduino. Quando o valor lido pelo LDR ultrapassa um determinado limiar, o LED acende.

Vídeo do Circuito com LDR

Código Utilizado no Arduino

Segue abaixo o código utilizado para a montagem do circuito com LDR e LED:

/***************************
 * Edusesc Taguatinga      *

 * Itinerário de robótica  *
 **************************/

// Declaração de Variáveis e Constantes
const int LED = 13;
int LDR = A0;

// Configurações
void setup() {
  pinMode(LED, OUTPUT);
  pinMode(LDR, INPUT);
  Serial.begin(9600);
}

// Repetições
void loop() {
  int luz = analogRead(LDR);
  Serial.println(luz);

  if (luz > 500) {
    digitalWrite(LED, HIGH);
  } else {
    digitalWrite(LED, LOW);
  }
}
